Project 01 — Featured

Student Success Early Warning System

An applied machine learning system that identifies academically at-risk students before final outcomes occur — enabling earlier, more targeted interventions. Built on the UCI Student Performance Dataset using Python and Scikit-learn.

→ Prior failures proved the strongest predictor of future academic risk
→ High absenteeism consistently correlated with lower success rates
→ Models identified at-risk students before final grades were recorded
